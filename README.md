# 🛍️ Ecommerce Fullstack Website

A complete MERN stack ecommerce application with user authentication, product management, cart, order placement, and payment integration.

- **Backend:** Node.js, Express.js, MongoDB
- **Authentication:** JWT
- **Hosting:** Render
